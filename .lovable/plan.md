## Варіант A — мінімальні покращення блоку "Відеогалерея"

Працюємо тільки в межах двох файлів. Структура та дизайн секції залишаються тими ж — тільки технічні фікси. Якщо щось не сподобається — кнопка revert під відповіддю поверне все назад.

### Що змінюємо

**1. `src/components/home/VideoRecordings.tsx`**

- Прев'ю YouTube: `maxresdefault.jpg` → `hqdefault.jpg` (надійніше, завжди існує, легший — краще LCP).
- На `<img>` додаємо `loading="lazy"`, `decoding="async"`, `width={480}`, `height={360}` — щоб не було CLS.
- Картка-посилання: додаємо `aria-label={`Дивитись відео: ${video.title}`}` та оборачуємо дату в `<time dateTime="YYYY-MM-DD">` (для SEO/a11y).
- Кнопка play отримує `aria-hidden="true"` (декоративна, бо вся картка — посилання).
- Прибираємо подвійний hover на overlay — лишаємо плавний `group-hover` ефект на картці (легке потемніння + scale прев'ю).
- Виправляємо посилання на YouTube канал внизу — замість зламаного encoded-URL використовуємо коротке: `https://www.youtube.com/@nise.com.ua/streams` (підставити реальний handle каналу — підтвердьте, який саме, або перевіримо в існуючих посиланнях у Footer/інших файлах).
- Імпорт даних — з нового файлу `src/data/videos.ts` (див. п.2).

**2. Новий файл `src/data/videos.ts`**

- Виносимо масив `videoRecordings` з компонента в окремий файл-дані з типом `Video`:
  ```ts
  export interface Video {
    id: number;
    title: string;
    youtubeId: string;
    duration: string;
    date: string;        // "17.06.2025"
    isoDate: string;     // "2025-06-17" — для <time>
  }
  ```
- `thumbnailUrl` і `videoUrl` будуються з `youtubeId` хелперами в самому компоненті (менше дублювання, легше додавати нові відео).
- Це і є "контент-файл" — додати нове відео = одна правка одного файлу, без чіпання JSX.

### Що НЕ зачіпаємо

- Дизайн, верстку, кольори, сітку, `bg-gray-50`, заголовок секції.
- Інші секції/файли, маршрути, SEO/structured data.
- Поточний список з 3 відео (травень-червень 2025) — залишається той самий контент. Окремо потім можна оновити `videos.ts` свіжими записами без зміни коду.

### Уточнення

- Для коректного посилання "більше на YouTube каналі" — підкажіть правильний handle каналу (`@...`) або підтвердіть, що ОК взяти з Footer / іншого існуючого посилання в проекті.

### Як відкотити

Після застосування — кнопка revert під моєю відповіддю повертає `VideoRecordings.tsx` і прибирає `src/data/videos.ts`. Один клік.
